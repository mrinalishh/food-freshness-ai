# Food Freshness AI

AI-powered web application that analyzes food images and predicts whether the item is **Fresh**, **Okay**, or **Spoiled**.

The system uses transformer-based computer vision models and provides results through a simple, user-friendly Flask interface.

---

## Features

- Upload food or fruit images  
- AI freshness prediction  
- Clean and minimal UI  
- Image preview after analysis  

---

## Tech Stack

- Python  
- Flask  
- Hugging Face Transformers  
- PyTorch  

---
